┌──────────────────────────────────────────────────────────────────────────┐
│                          📊 ARCHITECTURE                                  │
└──────────────────────────────────────────────────────────────────────────┘

 User Opens Jobs Page
         ↓
 Frontend (React) Requests Jobs
         ↓
 Job Agent Service (FastAPI)
         ↓
 Reads User's Roadmap from MongoDB
         ↓
 Fetches Jobs from APIs:
   1. RapidAPI (LinkedIn/Indeed)
   2. Adzuna (Global Jobs)
   3. AI Generation (Fallback)
         ↓
 Groq AI Scores Each Job (0-100%)
         ↓
 Cache in MongoDB (6 hours)
         ↓
 Return Sorted by Match Score
         ↓
 Frontend Displays with UI
